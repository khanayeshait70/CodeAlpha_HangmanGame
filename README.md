🎮 Hangman Game

📌 Description:
This is a simple Hangman game where the player has to guess the hidden word by suggesting letters.
Each wrong guess brings the hangman closer to completion. The game ends when the word is guessed correctly or attempts run out.

🎯 How to Play:
A random word is selected by the system
Player guesses one letter at a time
Correct guesses reveal letters in the word
Wrong guesses reduce remaining attempts
Win by guessing the full word before attempts finish

✨ Features:
Random word selection
Limited attempts system
Win/Lose conditions
Simple and beginner-friendly logic
